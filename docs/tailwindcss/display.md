# 布局相关

```css
.min-w-max {
    min-width: max-content;
}
.w-\[816px\] {
    width: 816px;
}
.flex {
    display: flex;
}
/* x轴自动居中 */
.mx-auto {
    margin-left: auto;
    margin-right: auto;
}
.justify-center {
    justify-content: center;
}
/* y轴上下间距1rem */
.py-4 {
    padding-top: 1rem;
    padding-bottom: 1rem;
}
/* 宽度填充满 */
.size-full {
    width: 100%;
    height: 100%;
}
/* x轴 */
.overflow-x-auto {
    overflow-x: auto;
}
```
